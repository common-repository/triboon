By using Triboon plugin make your way easier for managing your reportages from Triboon.net

== Description ==
This is a plugin that shows a list of reportages came from triboon with different states and website admin can preview, edit, accept, reject and publish website's related reportages with a lot of useful additions.

== Installation ==
First you should download and install the plugin, then enter your api-key in options page. your reportages will be shown in Triboon reportages page.
